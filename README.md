# Dark Mode Toggle

A simple front-end project that lets users switch between light mode and dark mode with a button click.

## Features

- Toggle between light and dark themes
- Smooth transition effect
- Clean centered layout
- Built with plain HTML, CSS, and JavaScript

## Project Structure

```text
Dark-Mode-Toggle/
|- index.html
|- style.css
|- main.js
|- README.md
```

## How It Works

- `index.html` creates the page layout and toggle button
- `style.css` defines the default light theme and the `.dark` theme styles
- `main.js` listens for the button click and toggles the `dark` class on the `body`

## Run Locally

1. Download or clone the project.
2. Open `index.html` in your browser.

No build tools or installation steps are required.

## Technologies Used

- HTML5
- CSS3
- JavaScript

## Preview

Click the **Toggle Dark Mode** button to switch the page theme instantly.
